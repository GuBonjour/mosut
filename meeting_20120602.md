  * 寫起來像集合定義

  * Prelude> :t
  * typeclass
  * Haskell 版本的 quicksort
```
qsort :: (Ord a) => [a] -> [a]
qsort []
qsort (x:xs) = qsort less ++ [x] ++ qsort more
    where less = filter (<x )  xs
    where more = filter (>=x)  xs
```
