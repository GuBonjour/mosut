# 簡介 #

邀約開放原始碼的同好，一起在充滿陽光與古色古香的府城，關注以下項目 (包含但不限)：
  * 開放標準 / 開放規格
  * 自由軟體 / 開放原始碼
  * 開放硬體
加入位於 Facebook 的粉絲團，以獲得最新的活動資訊並和成員們互動：https://www.facebook.com/groups/mosut/

# 初步想法 #

  * 南台灣的 open source 同好聚會，每兩周舉辦一次
  * 提供大學院校、公司團體，以及個人作技術討論交流的機會
  * 每次聚會固定作一項心得分享，並由與會者補充或進行個別討論
  * 增加與北部地區的社群互動
